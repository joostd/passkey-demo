Enterprise Attestation Tester

This page tests if your browser and FIDO security key support Enterprise Attestation.
  1. Use a FIDO2 security key that supports FIDO Enterprise Attestation.
  2. Set the origin for allowing enterprise attestation to
When using Chrome, enable chrome://flags/#web-authentication-permit-enterprise-attestation to permit a set of origins to request an attestation statement from a security key when creating a Web Authentication credential.
