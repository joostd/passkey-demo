Secure Payment Confirmation Tester

Simple (same-origin) demo page to test if your browser and FIDO Authenticator support Secure Payment Confirmation (SPC).
  1. Use a client that supports SPC, such as a recent Chrome browser
  2. Enable Secure Payment Confirmation Debug Mode to use a FIDO security key that supports the CTAP 2.2 thirdPartyPayment extension. See chrome://flags/#enable-debug-for-secure-payment-confirmation
  3. Optionally, enable Secure Payment Confirmation Network and Issuer Icons. See chrome://flags#enable-network-and-issuer-icons-for-secure-payment-confirmation
  4. When done, delete payment credentials from chrome://settings/passkeys
